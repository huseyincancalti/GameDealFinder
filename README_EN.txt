# GameDealFinder

**A simple price comparison program developed with C++**  
Compares game prices and shows the cheapest option.  
Supports adding, deleting, and updating game prices.

---

## 🚀 Features

- Compare game prices  
- Add new games  
- Delete existing games  
- Update game prices  
- Data is stored in `.txt` files (one per platform)

---

## 🛠 Installation

> **Note:** It is recommended to run the program as **Administrator** the first time.

1. Run the `GameDealFinder_Setup.exe` file  
2. Complete the installation  
3. Start the program from your desktop or the start menu

---

## 📌 Usage

### ▶️ Starting the Program

1. Open the **GameDealFinder** application  
2. Choose one of the following options from the main menu:
   - `1` – Search for a game
   - `2` – List all games
   - `3` – Add a new game
   - `4` – Delete a game
   - `5` – Update game price
   - `6` – Exit

### 📂 Data Storage

- All game data is stored in the `data/` folder  
- Each platform has its own `.txt` file
